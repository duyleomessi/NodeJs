This is small application to get the weather of the area.
To use it type.

It has 2 module: coordinate and weather.

Coordinate use to get the latitude and longitude of the place we want to get the weather.
Weather use to get weather.

node app.js (area you want to get the weather)

ex:1. node app.js Hoang Hoa, Thanh Hoa
   2. node app.js Ha Noi
